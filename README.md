# CodeAlpha Internship Tasks

## 📌 Overview

This repository contains the projects completed during the CodeAlpha Internship Program. The projects demonstrate skills in Python programming, data analysis, web scraping, and data visualization.

---

# 1. Crime Analytics Command Center

## Description

Crime Analytics Command Center is a Python-based dashboard that helps analyze and visualize crime-related data. It provides insights into crime trends, categories, and statistics through an interactive interface.

## Features

* Crime data analysis
* Interactive dashboard
* Statistical summaries
* Data visualization charts
* User-friendly interface

## Technologies Used

* Python
* Pandas
* Matplotlib
* Tkinter

---

# 2. Crime News Scraper

## Description

Crime News Scraper is a web scraping application that collects the latest crime-related news articles from online sources and displays them in a structured format.

## Features

* Fetches latest crime news
* Extracts headlines and links
* Organized news display
* Easy-to-use interface

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas

---

# 3. Netflix EDA (Exploratory Data Analysis)

## Description

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends, patterns, and insights related to movies and TV shows available on Netflix.

## Features

* Data cleaning and preprocessing
* Visualization of content trends
* Analysis by country, genre, and release year
* Insights through charts and graphs

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the desired project:

   ```bash
   python main.py
   ```
