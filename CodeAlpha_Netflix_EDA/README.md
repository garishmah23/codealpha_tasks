# Netflix Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset. The goal is to explore the dataset, understand its structure, identify trends and patterns, and generate meaningful insights through data analysis.

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

---

## 🎯 Objectives

* Explore the Netflix dataset structure
* Handle missing values
* Analyze Movies vs TV Shows
* Identify top content-producing countries
* Analyze content ratings distribution
* Explore release year trends
* Generate insights from the data

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Analysis Performed

### 1. Dataset Exploration

* Dataset shape
* Column names
* Data types
* Missing values

### 2. Data Cleaning

* Identified missing values
* Replaced null values with appropriate placeholders

### 3. Movies vs TV Shows Analysis

* Counted total Movies
* Counted total TV Shows
* Compared content distribution

### 4. Country Analysis

* Identified top countries producing Netflix content
* Generated country-wise statistics

### 5. Ratings Analysis

* Analyzed content ratings
* Identified the most common rating categories

### 6. Release Year Analysis

* Explored content distribution by release year
* Identified popular release periods
